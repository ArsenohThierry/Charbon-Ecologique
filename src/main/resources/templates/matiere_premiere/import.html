Import CSV

Importer des types de matières premières depuis un fichier CSV.

Format requis

Colonne Type Contraintes Exemple
reference Texte Obligatoire, unique, max 50 car. CHARB-001
libelle Texte Obligatoire, max 150 car. Charbon de bois premium
prix_unitaire Nombre Obligatoire, > 0, max 2 décimales 15000.00
fournisseur Texte Email ou nom du fournisseur existant contact@andry.mg
actif Booléen true/false, 1/0, oui/non true

# Exemple de fichier CSV :

reference,libelle,prix_unitaire,fournisseur,actif

CHARB-001,Charbon de bois premium,15000.00,contact@andry.mg,true

BOIS-002,Bois de chauffage,8500.50,info@charbon.mg,false
